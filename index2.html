Mercado Libre

¿Dónde conseguirlo? Entrá a developers.mercadolibre.com.ar → tu app → Credenciales. El Seller ID es tu User ID de la cuenta PARKA (visible en api.mercadolibre.com/users/me con el token activo).

Tienda Nube
Sin configurar
¿Dónde conseguirlo? TN Admin → Configuración → Aplicaciones y canales de venta → Crear app privada → generá el token con permiso read_orders. El Store ID es el número en tu URL de admin.
